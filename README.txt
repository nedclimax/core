Core C libraries

- Modules:
  - ./base/     Basic functionality used everywhere. Every module depends on it.
  - ./os/       Operating system layer
  - ./build/    Compile C/C++ projects using C/C++
  - ./ui/       Immediate mode UI library
  - ./gfx/      Simple 2D cross platform graphics api
  - ./gpu/      Platform agnostic interface for talking to the gpu
  - ./math/     Advanced mathematics library, includes linear algebra and quaternions.
